## Selected Builds

### **creava-platform**
> クリエイター向け Portfolio / Fanclub / Store を横断するプラットフォーム。  
React + Vite フロント、Strapi v5、モノレポ構成で、UI と CMS 連携、認証、決済導線、CI/CD まで一貫して扱う前提の設計。

### **lunaria**
> Discord Bot・管理ダッシュボード・API・Worker を分離したコミュニティ運営基盤。  
Bot platform / Rule engine / Dashboard を個別に成立させつつ、モノレポで統合運用する設計の実験場。

### **vaultsend**
> Go + chi + PostgreSQL の backend foundation。  
フロント実装だけではなく、API 設計・永続化・運用を見据えた土台づくりまで担当できることを示すリポジトリ。

### **SkillTree-Forge**
> ブラウザ上で使う軽量ツール / エディタ系プロジェクト。  
React + TypeScript + Vite をベースに、SVG やグラフ UI を扱う“操作体験”を重視した実装。
